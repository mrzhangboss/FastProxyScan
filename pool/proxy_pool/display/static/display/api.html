API文档说明 Django Cache

使用Django的缓存技术，将最新的100个可用代理返回，每20分钟更新一次

接口详情

URL
/display/api
方式
GET
返回
Json
字段介绍
      ip： ip地址
      port： 端口
      protocol: 协议（http|https|http+https|socks)
      speed: 速度(秒）
      checked_state: 代理类型(未知, 'Default'),(透明代理, 'TransparentProxy'),(匿名代理, 'AnonymousProxy'),(高匿代理, 'HighAnonymousProxy'),(需要账号密码代理, 'NeedAuthProxy'),(中间人代理, 'MitmProxy'))
      insert_at: 入库时间
      update_at: 更新时间
